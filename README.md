# 🤖 AI FAQ Chatbot

An intelligent Customer Support FAQ Chatbot built using **Natural Language Processing (NLP)**, **TF-IDF Vectorization**, **Cosine Similarity**, and **Streamlit**.

This chatbot retrieves the most relevant answer from a FAQ knowledge base based on the user's query.

---

## 📌 Features

* Intelligent FAQ Retrieval
* NLP Text Preprocessing
* TF-IDF Vectorization
* Cosine Similarity Search
* Interactive Streamlit Web Application
* Conversation History
* Similarity Score Display
* Clean and User-Friendly Interface

---

## 🛠 Technologies Used

* Python
* Streamlit
* Scikit-learn
* Pandas
* NumPy
* Joblib
* NLP
* TF-IDF
* Cosine Similarity

---

## 📂 Dataset

Customer Support FAQ Dataset

* 500 Frequently Asked Questions
* Categories:

  * Account Management
  * Password Reset
  * Orders
  * Shipping
  * Payments
  * Refunds
  * Customer Support

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/CodeAlpha_FAQ_Chatbot.git
```

Move into the project folder

```bash
cd CodeAlpha_FAQ_Chatbot
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
